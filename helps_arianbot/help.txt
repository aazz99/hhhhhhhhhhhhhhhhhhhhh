➖ (ذکر)  = ارسال ذکر روز
➖ (نام شاخ)  = ارسال نام شاخ
➖ (حدیث)  = ارسال حدیث
➖ (لینک)  = نمایش لینک گروه
➖ (افزودن) یا (add/)  = افزودن کاربر به گروه با [ آیدی - ریپلای ](بیو)  = ارسال حدیث
➖ (بیو)  = ارسال بیو
➖ (حدیث)  = ارسال حدیث
➖ (داستان)  = ارسال داستان
➖ (قوانین)  = ارسال قوانین گروه
➖ (سرگرمی)  = ارسال بازی در گروه و با فرستان کلمه (یپرس) ربات از شما سوال میپرسه
➖ (ساعت)  = ارسال ساعت
➖ (عید)  = ارسال تاریخ عیذ
➖ (جوک)  = ارسال جوک در گروه

➖ (قطع کال)  = قعط کردن کال

➖ (امتیاز)  = امتیاز دادن به کار
➖ (کسر امتیاز)  = حذف امتیاز کاربر

➖ (ارتقا)  = ادمین کردن کاربر
➖ (تنزیل)  = حذف ادمین
